# OSS Indonesia Review Scraper

Proyek untuk scraping review aplikasi OSS Indonesia dari Google Play Store.

## Fitur
- Mengambil data: nama pengguna, ulasan, rating, tanggal review
- Menyimpan data dalam format JSON dan CSV
- Analisis statistik sederhana
- Pagination otomatis untuk mengambil banyak review

## Instalasi
1. Clone repository ini
2. Install dependencies: `pip install -r requirements.txt`

## Penggunaan
Jalankan script utama:
```bash
python scraper.py
